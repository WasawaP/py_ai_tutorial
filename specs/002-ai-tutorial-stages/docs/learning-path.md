# 学习路线图 (Learning Path)

本教程采用**渐进式、实战驱动**的学习路径,从传统机器学习到深度学习,最后到大语言模型应用。每个阶段都包含理论学习、动手实践和真实项目。

---

## 🎯 阶段3: 机器学习与数据挖掘 (Stage 3)

**目标**: 掌握传统机器学习算法,能够处理结构化数据并构建预测模型

**预计时间**: 
- 核心学习: 4-6小时
- 项目实践: 2-4周(每周投入5-10小时)

**适合人群**: 
- ✅ 有Python基础的开发者
- ✅ 想快速应用AI到业务的工程师
- ✅ 需要处理表格数据和做预测的数据分析师

### Module M01: 科学计算基础 (2-3小时)

**学习目标**:
- 熟练使用NumPy进行数值计算
- 掌握Pandas的数据操作和分析
- 能够用Matplotlib/Seaborn创建可视化图表

**学习内容**:
1. NumPy数组操作 (索引、切片、广播)
2. Pandas DataFrame操作 (筛选、分组、聚合)
3. 数据可视化 (折线图、散点图、热力图、箱线图)

**里程碑检查**:
- [ ] 能够独立读取CSV并进行基本数据清洗
- [ ] 能够创建至少3种不同类型的图表
- [ ] 理解GroupBy和透视表的使用场景

**练习建议**:
- 在[Kaggle](https://www.kaggle.com/)上找一个小数据集练习EDA
- 尝试复现《Python for Data Analysis》书中的示例

### Module M02: Pandas项目实战 (1-2小时)

**学习目标**:
- 掌握数据预处理全流程
- 学会特征工程的常用技巧
- 理解数据合并和缺失值处理

**学习内容**:
1. 数据清洗 (缺失值、异常值、重复值)
2. 特征编码 (Label Encoding, One-Hot, Frequency)
3. 特征缩放 (标准化、归一化)
4. 数据合并 (merge, join, concat)

**里程碑检查**:
- [ ] 能够识别并处理各类数据质量问题
- [ ] 能够将类别特征转换为数值特征
- [ ] 理解特征工程对模型性能的影响

**练习建议**:
- 处理Titanic数据集(经典入门数据集)
- 完成本模块的配套Notebook练习

### Module M03: AI数学基础 (1-2小时)

**学习目标**:
- 理解机器学习的数学直觉
- 掌握概率统计基本概念
- 了解线性代数在ML中的应用

**学习内容**:
1. 概率与统计 (正态分布、假设检验、中心极限定理)
2. 线性代数 (向量、矩阵、特征值、PCA)
3. 优化基础 (梯度、导数、梯度下降)

**里程碑检查**:
- [ ] 能够解释正态分布和假设检验的含义
- [ ] 理解PCA降维的原理和应用场景
- [ ] 能够用Python实现简单的梯度下降

**练习建议**:
- 观看3Blue1Brown的线性代数系列视频
- 用NumPy手动实现PCA算法

### Module M04: 机器学习进阶 (2-3小时)

**学习目标**:
- 掌握主流监督学习算法
- 理解模型评估和选择方法
- 学会超参数调优技巧

**学习内容**:
1. 回归算法 (线性回归、岭回归、多项式回归)
2. 分类算法 (逻辑回归、决策树、随机森林、XGBoost)
3. 聚类算法 (K-means、层次聚类)
4. 模型评估 (交叉验证、混淆矩阵、ROC/AUC)
5. 超参数调优 (网格搜索、随机搜索)

**里程碑检查**:
- [ ] 能够选择合适的算法解决实际问题
- [ ] 理解偏差-方差权衡和过拟合/欠拟合
- [ ] 能够使用scikit-learn完成完整的建模流程

**练习建议**:
- 参加Kaggle入门竞赛(如Titanic, House Prices)
- 完成本模块的8个算法实践Notebook

### 阶段3实战项目 (2-4周)

完成至少**3个项目**(推荐顺序):

**优先级P1** (必做,打好基础):
1. **医疗数据分析** (P01-Healthcare) - 糖尿病风险预测
2. **电商用户画像** (P02-Ecommerce) - 用户分群与RFM分析
3. **金融风控** (P03-Finance) - 信用卡欺诈检测

**优先级P2** (巩固提升):
4. **通信行业** (P04-Telecom) - 客户流失预测
5. **零售优化** (P05-Retail) - 销售预测与库存管理
6. **互联网推荐** (P06-Internet) - 协同过滤推荐系统

**优先级P3** (进阶挑战):
7. **电商年度复盘** (P07-Ecommerce-Annual) - 多维度数据分析
8. **航空公司** (P08-Airline) - 旅客价值分析
9. **信贷评估** (P09-Credit) - 贷款违约预测

**每个项目包含**:
- 完整的Jupyter Notebook (含详细注释)
- 真实或仿真的业务数据
- 从EDA到建模的完整流程
- 可复现的结果和评估报告

**阶段3完成标准**:
- ✅ 完成所有Module的学习
- ✅ 至少完成3个项目(P1优先)
- ✅ 能够独立完成一个端到端的ML项目
- ✅ 理解数据预处理→特征工程→建模→评估的完整流程

## 🚀 阶段4: 深度学习 (Stage 4)

**目标**: 掌握深度学习基础理论和主流框架,能够训练CNN/RNN模型

**预计时间**: 
- 核心学习: 6-10小时
- 项目实践: 1-2个月(每周投入5-10小时)

**适合人群**: 
- ✅ 已完成阶段3的学习者
- ✅ 需要处理图像/文本/序列数据的工程师
- ✅ 想要了解神经网络原理的开发者

### Module M05: 深度学习基础 (2-3小时)

**学习目标**:
- 理解神经网络的工作原理
- 掌握反向传播和优化算法
- 学会使用PyTorch/TensorFlow

**学习内容**:
1. 神经网络基础 (感知机、多层感知机、激活函数)
2. 反向传播算法 (链式法则、梯度计算)
3. 优化器 (SGD、Adam、学习率调度)
4. 正则化技术 (Dropout、Batch Normalization、数据增强)
5. PyTorch/TensorFlow框架基础

**里程碑检查**:
- [ ] 能够手动实现前向传播和反向传播
- [ ] 理解常用激活函数和优化器的特点
- [ ] 能够用PyTorch/TensorFlow训练简单的神经网络

**练习建议**:
- 用PyTorch实现MNIST手写数字识别
- 可视化神经网络的训练过程

### Module M06: 计算机视觉 (2-4小时)

**学习目标**:
- 掌握CNN的原理和应用
- 学会使用预训练模型
- 了解目标检测和图像分割

**学习内容**:
1. CNN基础 (卷积层、池化层、全连接层)
2. 经典架构 (LeNet、AlexNet、VGG、ResNet)
3. 目标检测 (YOLO、Faster R-CNN)
4. 图像分割 (U-Net、Mask R-CNN)
5. 迁移学习和微调

**里程碑检查**:
- [ ] 能够解释卷积层如何提取特征
- [ ] 能够使用预训练模型进行图像分类
- [ ] 理解目标检测和分类的区别

**练习建议**:
- 训练一个猫狗分类器
- 使用YOLOv8进行实时目标检测

### Module M07: 自然语言处理 (2-3小时)

**学习目标**:
- 理解RNN和Transformer架构
- 掌握词嵌入和序列建模
- 学会使用预训练语言模型

**学习内容**:
1. 文本预处理 (分词、词向量、Word2Vec)
2. RNN家族 (RNN、LSTM、GRU)
3. Transformer架构 (Self-Attention、多头注意力)
4. 预训练模型 (BERT、GPT)
5. NLP任务 (文本分类、命名实体识别、机器翻译)

**里程碑检查**:
- [ ] 理解RNN如何处理序列数据
- [ ] 能够解释Transformer的注意力机制
- [ ] 能够使用HuggingFace加载和微调预训练模型

**练习建议**:
- 用LSTM进行情感分析
- 用BERT进行文本分类

### 阶段4实战项目 (1-2个月)

完成至少**2个项目**(1个CV + 1个NLP):

**计算机视觉项目**:
1. **工业视觉检测** (P01-Industrial-Vision) - 产品缺陷检测
2. **YOLOv11实时检测** (P02-YOLOv11) - 实时目标检测系统
3. **OCR文字识别** (P03-OCR) - 票据文字提取
4. **图像分割** (P04-Segmentation) - 医学影像分割
5. **医学影像分析** (P05-Medical-Imaging) - X光/CT图像分类

**NLP项目**:
6. **Transformer翻译** (P06-Translation) - 英中机器翻译
7. **预训练模型信息提取** (P07-Info-Extraction) - 命名实体识别

**阶段4完成标准**:
- ✅ 完成所有Module的学习
- ✅ 至少完成2个项目(1 CV + 1 NLP)
- ✅ 能够训练和部署深度学习模型
- ✅ 理解CNN/RNN/Transformer的原理和应用场景

## 🤖 阶段5: AIGC与大模型 (Stage 5)

**目标**: 掌握大语言模型的使用和微调,能够开发LLM应用

**预计时间**: 
- 核心学习: 1-2天
- 项目实践: 1-2周

**适合人群**: 
- ✅ 已完成阶段3(或阶段4)的学习者
- ✅ 需要集成LLM到产品的工程师
- ✅ 想要了解ChatGPT背后技术的开发者

### Module M08: LLM原理与应用 (4-6小时)

**学习目标**:
- 理解大语言模型的工作原理
- 掌握Prompt Engineering技巧
- 学会使用LangChain构建应用

**学习内容**:
1. LLM概述 (GPT、DeepSeek、LLaMA)
2. Prompt Engineering (Few-shot、Chain-of-Thought)
3. RAG (检索增强生成)
4. Agent (自主智能体)
5. LangChain框架

**里程碑检查**:
- [ ] 能够编写有效的Prompt获取期望输出
- [ ] 理解RAG如何增强LLM的知识
- [ ] 能够使用LangChain构建简单应用

**练习建议**:
- 设计不同场景的Prompt模板
- 构建一个简单的知识库问答系统

### Module M09: 大模型开发 (4-8小时)

**学习目标**:
- 掌握大模型的微调方法
- 了解模型部署和优化技巧
- 学会评估LLM应用的性能

**学习内容**:
1. Fine-tuning技术 (全量微调、LoRA、QLoRA)
2. 模型部署 (FastAPI、vLLM、量化)
3. 评估与监控 (困惑度、人工评估、A/B测试)
4. 安全与伦理 (内容过滤、偏见检测)

**里程碑检查**:
- [ ] 能够使用LoRA微调开源模型
- [ ] 理解模型量化和加速技术
- [ ] 能够设计LLM应用的评估指标

**练习建议**:
- 微调一个小型LLaMA模型
- 部署一个本地LLM推理服务

### 阶段5综合项目 (1-2周)

**对话系统端到端实现** (P01-Dialogue-System):
- 需求分析与架构设计
- 知识库构建与向量检索
- LLM集成与Prompt优化
- 对话管理与上下文维护
- API开发与前端交互
- 性能优化与部署

**阶段5完成标准**:
- ✅ 完成所有Module的学习
- ✅ 完成对话系统综合项目
- ✅ 能够独立开发和部署LLM应用
- ✅ 理解RAG、Agent、Fine-tuning的应用场景

## 📈 学习时间线

### 快速通道 (2-3个月)

**适合**: 全职学习或每周投入20+小时

- **Week 1-2**: 阶段3 Module M01-M04 + 3个必做项目
- **Week 3-4**: 阶段4 Module M05-M06 + 1个CV项目
- **Week 5-6**: 阶段4 Module M07 + 1个NLP项目
- **Week 7-8**: 阶段5 Module M08-M09 + 对话系统项目
- **Week 9-12**: 深入练习、参加竞赛、构建个人项目

### 标准通道 (3-6个月)

**适合**: 在职学习,每周投入10-15小时

- **Month 1**: 阶段3完整学习(所有Module + 至少3个项目)
- **Month 2-3**: 阶段4完整学习(所有Module + 至少2个项目)
- **Month 4**: 阶段5完整学习(所有Module + 对话系统)
- **Month 5-6**: 巩固提升,完成更多项目,准备面试/转岗

### 进阶通道 (6-12个月)

**适合**: 每周投入5-10小时,稳扎稳打

- **Month 1-2**: 阶段3 (深入理解每个算法,完成所有9个项目)
- **Month 3-6**: 阶段4 (系统学习DL,完成所有7个项目)
- **Month 7-8**: 阶段5 (LLM应用开发)
- **Month 9-12**: 参加Kaggle竞赛,构建完整的产品级项目,准备简历

## 💡 学习建议

### 1. 循序渐进,不要跳过基础
- 阶段3是基础,必须扎实掌握
- 阶段4可以先学CV或NLP,根据兴趣选择
- 阶段5依赖阶段3的数据处理能力

### 2. 动手实践,多写代码
- 每个概念都要亲自写代码验证
- 不要只看教程,要独立完成项目
- 遇到问题先自己调试,再查资料

### 3. 建立知识体系,做好笔记
- 使用思维导图整理知识点
- 记录常见错误和解决方案
- 定期复习和总结

### 4. 参与社区,寻求反馈
- 在GitHub上分享你的项目
- 参加Kaggle竞赛获得实战经验
- 加入学习社群互相交流

### 5. 持续学习,保持更新
- AI领域发展快,定期学习新技术
- 关注顶会论文(NeurIPS、ICML、CVPR)
- 尝试复现最新的研究成果
